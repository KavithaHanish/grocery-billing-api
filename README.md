# Grocery Billing API

A simple grocery and vegetable billing system built using ASP.NET Core Web API and SQL Server.

## Features
- Add grocery products
- Manage vegetable prices per kg
- Stock management
- Generate bills
- Send bills via WhatsApp

## Technology Stack
- ASP.NET Core Web API
- Entity Framework Core
- SQL Server
- Angular (frontend planned)

## Use Case
Designed for small grocery and vegetable shops to easily generate bills based on weight (grams / kg).
